# Sử dụng GIT cơ bản

-   Tạo tài khoản github.com
-   Cài đặt Git scm: https://git-scm.com/
-   Kiểm tra:

*   Mở terminal
*   Gõ: git -v

Các bước đẩy code lên git

Gõ lần các câu lệnh sau

git add .
git commit -m 'noi dung'
git push

Các bước lấy code từ repository về máy

Nếu lần đầu: git clone link_repo
